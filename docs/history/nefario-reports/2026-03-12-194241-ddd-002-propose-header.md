## Key Design Decisions

1. **Stacked three-line layout**: "Mostly" (small, regular) above "Hallucinations" (large, semibold) above tagline (small, italic, muted). Left-aligned to content column edge.

2. **Scroll-away header**: position: relative at all viewports. No fixed header — the site has no navigation items warranting persistent presence. Reclaims 80px of mobile reading space.

3. **Corrupted letterforms**: 5 of 14 letters in "Hallucinations" carry subtle corruptions (stroke overshoot, counter closure, phantom serif, asymmetric crossbar, broken junction). Corruption density increases left-to-right, mimicking AI text degradation. Map marked as illustrative.

4. **Dual implementation paths**: CSS-only (per-letter spans with transforms) recommended first. SVG fallback (modified glyph paths) for full fidelity. DDD specifies visual outcome, defers implementation choice.

5. **Fluid typography**: clamp(28px, 8vw, 42px) for "Hallucinations" avoids discrete breakpoint jumps. min-height replaces fixed height for nav container.

6. **Tagline in Source Serif 4 italic**: Register shift from monospaced to serif signals editorial voice. Subordinated via --color-text-muted.
